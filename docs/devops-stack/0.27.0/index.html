Camptocamp’s DevOps Stack

The DevOps Stack provides a standard Kubernetes Platform, bringing together automated Provisioning as Code, Continuous Application Deployment, and readily-made configurations for the best tools in the industry.

The easiest wat to get started with the DevOps Stack is definitely to follow the K3s on Docker Quickstart. This way you will have a fully operational K3s playground on your local machine.

Project sponsor

This project is mainly developed and sponsored by Camptocamp. The source code is open, distributed under the Apache 2 license and available on GitHub.
