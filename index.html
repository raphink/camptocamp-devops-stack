Stack Components

Kubernetes

At the heart of the DevOps Stack, Kubernetes powers all application deployments, and more.

Terraform

The base layer of the Stack, provisioning the whole Infrastructure as Code, from IaaS to Argo CD.

Argo CD

DevOps Stack Applications are deployed in a GitOps fashion, using Continuous Deployment for Kubernetes Resources.

Prometheus

The standard in Kubernetes Time Series Database, set up for the whole Stack out-of-the-box.

Grafana

Readily-made Observability Dashboards are provided for the Kubernetes cluster and Stack Components.

Loki

Application logs are gathered and made accessible using Loki inside Grafana.

Cert-Manager

Valid public certificates are automatically provided for all Stack Components, courtesy of Let’s Encrypt.

Open ID Connect

Authentication is set up for all Components in the Stack, using either the platform’s authentication system, or Keycloak.

Traefik

The Traefik Ingress controller is used to route web access to all Components, on your domain of choice.
